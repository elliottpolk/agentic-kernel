**Operator request:** Add a `.agentic/instructions/` extension point mirroring `.github/instructions/` semantics. Update all kernel files to reflect this. Extend `copilot-bridge` to bridge instruction files to `.github/instructions/`.

- Model: exact mirror of `.github/instructions/`. Files are Markdown with optional `applyTo` frontmatter (glob). Global instructions (no `applyTo`) load at init. Scoped instructions apply when working on matching files.
